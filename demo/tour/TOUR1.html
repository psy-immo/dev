Tour durch die Versuchsschnittstelle

Bevor der eigentliche Versuch startet, bitten wir Sie, sich mit der Umgebung vertraut zu machen.

Argument zusammenstellen

Um ein Argument zusammenzusetzen, müssen alle Lücken mit Satzteilen gefüllt werden, sowie gegebenenfalls ein Element einer Mehrfachauswahl gewählt werden. Die für jede Lücke zur Verfügung stehenden Satzteile sind am linken Rand dargestellt, die Hintergrundfarbe gibt hierbei einen Hinweis darauf, an welchen Stellen der jeweilige Teil Verwendung finden kann. Um eine Lücke zu füllen, klicken Sie auf das gewünschte Fragment am linken Rand, um es aufzuheben. Jetzt wird das gewählte Fragment neben dem Mauscursor dargestellt. Danach klicken Sie auf die Lücke im Argumentbaukasten, die Sie füllen oder ändern wollen, um das Fragment dort abzulegen.
Auf die selbe Art und Weise können Sie die Satzteile aus den Lücken im Argumentbaukasten wieder aufnehmen, falls Sie einen Satzteil doch nicht mehr verwenden wollen, können Sie diesen auch in den großen Mülleimer links neben den Argumentekasten ablegen.
Eine Mehrfachauswahl kann durch einfaches klicken auf die gewünschte Textvariante getätigt werden. Sollte noch kein Element ausgewählt worden sein, ist der Hintergrund aller Elemente der Mehrfachauswahl grau. Ansonsten ist der Hintergrund der nicht gewählten Varianten hellblau und der Hintergrund der gewählten Variante dunkenblau.

Argumente auswählen

Wenn ein Argumentschema vollständig ist, das heisst, dass sämtliche Lücken gefüllt worden sind und jede Mehrfachauswahl ein dunkelblau markiertes Element besitzt, ändert sich die Hintergrundfarbe des Baukastens von hellblau auf dunkelblau. Der so gebildete Satz kann nun durch einfachen Klick auf das Flugzeugsymbol, welches sich direkt links vom ersten Satzteil befindet, angehoben werden. Danach kann es entweder innerhalb des hellblauen Rechtecks unter der Überschirft "Argumente" als Argument abgelegt werden, oder aber in den großen Mülleimer links neben dem Argumentekasten geworfen werden.
Falls Sie Argumente aus dem Argumentekasten entfernen möchten, können Sie dies durch einfaches Klicken auf den jeweiligen kleinen Papierkorb links neben dem Argument tun. Alternativ können Sie durch einfaches Klicken auf den Argumenttext selbiges wieder anheben und im großen Mülleimer ablegen.
